📌 Team Task Manager

A full-stack Team Task Management Web Application that allows users to manage projects, assign tasks, and collaborate efficiently.

🚀 Features
🔐 User Authentication (Login & Signup)
📋 Create & Manage Projects
✅ Task Creation, Assignment & Tracking
👥 Team Collaboration
📊 Dashboard with task overview
⚙️ User Settings
🔒 Protected Routes (Auth-based access)
🛠️ Tech Stack

Backend

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
2️⃣ Setup Backend
cd backend
npm install
npm start
3️⃣ Setup Frontend
cd frontend
npm install
npm run dev
🌐 Environment Variables

Create a .env file in backend:

PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
